# RGBA and HEX Color Generator üé®

Project of a color generator for RGBA and HEX colors created with HTML, CSS, and JavaScript.

## Resources Used

- HTML
- CSS
- JavaScript

## Basic Tutorial

Simply drag the sliders for each color to generate new colors. You can also adjust the transparency and save the RGBA code of the selected color. Additionally, the application allows you to generate HEX color codes and copy them.

## Generating HEX Color Code

In addition to RGBA colors, this application also generates HEX color codes. You can easily copy the generated HEX code for use in your project.
